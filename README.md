## 🚀 快速開始

### 環境要求
- Node.js 20.19.0 或以上
- npm 已安裝

### 本地開發

```bash
# 安裝依賴
npm install

# 啟動開發伺服器
npm run dev

# 編譯生產版本
npm run build

# 預覽編譯結果
npm run preview
```

### 部署到 GitHub Pages

```bash
# 一鍵部署
npm run deploy
```

部署詳細步驟與常見問題，請參考 [GH_PAGES_GUIDE.md](GH_PAGES_GUIDE.md)。

## 📁 專案結構

```
curriculum-vitae/
├── src/
│   ├── components/        # 可複用組件
│   ├── views/            # 頁面視圖
│   ├── data/             # 資料定義（portfolio.js）
│   ├── assets/           # 圖片、樣式等資源
│   └── main.js           # 應用入口
├── public/               # 靜態資源
├── vite.config.js        # Vite 配置
├── package.json          # 依賴與指令
└── README.md            # 本檔案
```

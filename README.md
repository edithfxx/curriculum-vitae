# 作品集網站 Portfolio

這是我的個人作品集網站，展示我在前端開發上的實作經驗、作品連結與技能優勢。

## 作品連結

1. Microsoft 仿切
   - 連結: https://edithfxx.github.io/personal-portfolio/microsoft-cut/index.html
   - 類型: 靜態切版 / RWD
   - 重點: HTML 結構規劃、CSS 版面控制、導覽列與卡片式排版

2. 九星連線
   - 連結: https://edithfxx.github.io/personal-portfolio/nine-stars/index.html
   - 類型: CSS 動畫 / 視覺特效
   - 重點: CSS 動畫、星體排列、視覺層次與動畫節奏

3. 行星繞日
   - 連結: https://edithfxx.github.io/personal-portfolio/planets-orbiting-the-sun/index.html
   - 類型: CSS 動畫 / 軌道模擬
   - 重點: CSS keyframes、定位、旋轉動畫與節奏控制

4. 個人速查表
   - 連結: https://edithfxx.github.io/cheat-sheet
   - 類型: Vue 3 CDN / 知識管理工具
   - 重點: CRUD、搜尋、高亮、JSON 匯入匯出、localStorage

5. 色弱遊戲
   - 連結: https://edithfxx.github.io/personal-portfolio/color-blindness/index.html
   - 類型: JavaScript 互動 / 小遊戲
   - 重點: 遊戲邏輯、DOM 操作、倒數計時、Web Audio API

6. 主題收藏展示館
   - 連結: https://edithfxx.github.io/personal-portfolio/collect/index.html
   - 類型: Vue 3 / 資料管理
   - 重點: 表單處理、分類篩選、localStorage、WCAG A / AA

7. 金門水產試驗所網站改版 Prototype
   - 連結: https://nchu-11503-kinmen-fishers.dev-hub.io/
   - 類型: 網站改版 / 官方形象網站
   - 重點: 資訊架構、Vue 3 前端介面、路由、狀態管理、雙語切換

## 專案啟動

```sh
npm install
npm run dev
```

> 此專案使用 Vue 3 與 Vite 建構，適合開發與快速部署。
